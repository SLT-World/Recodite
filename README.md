# Recodite
A fast, lightweight, cross-platform desktop local **media compressor** built with MAUI.

Recodite helps you shrink large files fast, keep your quality, and stay in control of your data. Everything runs locally on your device, so your files never leave your computer.

## Notable Features
- Local video compression
- Hardware acceleration
- Fully offline
- Drag & drop support
- Presets
- Format conversion
- Custom output path

## Installation
If you are running Windows 10 or above, Recodite can be downloaded with ease from the [Microsoft Store](https://apps.microsoft.com/detail/9n1m86937z91).

## Usage
1. Add one or more video files (+ button or drag & drop).
2. Pick a preset (optional).
3. Click **Compress**.
4. Wait for compression to finish.
5. Save the output.
