# Recodite
A cross-platform desktop **media compressor** built with MAUI.

> [!WARNING]  
> Recodite is a work in progress.
> Features and behavior may change at any time.

## Usage
1. Add one or more video files (+ button or drag & drop).
2. Click **Compress**.
3. Wait for compression to finish.
4. Save the output.

## Platforms
- Windows
- MacOS
